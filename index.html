DiagnoseNET

Is a development application-framework for tailoring deep neural networks to computational architectures with an efficient balance between accuracy and energy consumption.

DiagnoseNET is extending TensorFLow library to characterize the deep learning tasks and improve the balance between accuracy and energy-efficient performance. The first application built is for deploying medical diagnostic tools inside the hospitals...
